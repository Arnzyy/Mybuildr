# Golden Paths

These flows must ALWAYS work. Test after every ZIP.

## Path 1: Email Conversion
```
Land on page → Scroll to see value → Click email CTA → Email link opens
```

## Path 2: Demo Interaction
```
Scroll to demo section → Click first tab → See website preview
                      → Click second tab → See admin panel
                      → Click third tab → See Instagram feed
```

## Path 3: Pricing Discovery
```
Scroll to pricing → See £99 option → See £149 option → See £199 option
               → Understand what's included → Click email CTA on tier
```

## Path 4: Mobile Experience
```
Land on mobile → Hero is readable → Can scroll smoothly → CTAs are clickable
             → Demo tabs work → Pricing stacks properly → Email works
```

---

## Quick Test Template

After each ZIP:

- ✅ Path 1: Email CTA works on all sections
- ✅ Path 2: Demo tabs switch smoothly
- ✅ Path 3: Pricing tiers readable
- ✅ Path 4: Mobile UX is smooth and buttons are clickable

If ANY fails → STOP and fix before continuing.
